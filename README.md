## Downloads

Windows: None

## Basic Usage

```bash
ginger new MyProject # creates new project
ginger new MyProject --template git_repo_uri # use a templat
ginger lemon # starts lemon server
ginger run # runs program
ginger interact # opens interactive enviroment
```
